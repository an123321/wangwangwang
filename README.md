wangwang
=====

项目介绍（如果对你有用，请给个star！）
------
汪汪旺影院该网站是一个在线购买电影票系统，登录即可买票。致力于打造适用的分布式系统。采用前后端分离模式，前台采用vue.js为核心框架。


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
